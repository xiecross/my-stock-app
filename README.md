# 股票数据库目录

此目录用于存储本地股票数据库缓存文件。

## 文件说明

- `stock_list.json`: 股票代码和名称的本地数据库
  - 自动生成，每24小时更新一次
  - 包含所有A股股票的代码和名称
  - 用于加速股票搜索功能

## 初始化

首次使用时，可以运行初始化脚本预先生成数据库：

```bash
python init_stock_db.py
```

或者直接运行应用，系统会自动下载并创建数据库。

## 数据更新

- **自动更新**: 每24小时自动从网络更新
- **手动更新**: 在应用侧边栏点击"🔄 手动刷新数据库"按钮

## 注意事项

- 数据库文件大小约 1-2 MB
- 首次下载可能需要几秒钟
- 建议定期更新以获取最新的股票列表
